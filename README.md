# HelpDesk

HelpDesk is a web-based ticketing and support management system built with **ASP.NET Core (.NET)** for the backend and **Angular** for the frontend.

This project is designed to help users create, track, and manage support tickets more efficiently through a modern web application architecture.

---

## Tech Stack

### Backend
- ASP.NET Core / .NET
- Entity Framework Core
- SQL Server
- JWT Authentication

### Frontend
- Angular
- TypeScript
- HTML / CSS

---

## Project Structure

```bash
HelpDesk/

└── README.md

## Features

- User authentication and authorization
- Create support tickets
- View ticket details
- Update ticket status
- Add ticket comments
- Track ticket history
- Role-based access for users, admins, and staff
